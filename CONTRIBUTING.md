# Contributing

Thanks for your interest in contributing!

## How to contribute

1. Fork this repository.
2. Create a new branch for your changes.
3. Edit `skills/SKILL.md` or add new files as needed.
4. Submit a Pull Request with a clear description of what you changed and why.

## Guidelines

- Keep `skills/SKILL.md` focused on actionable rules, not general advice.
- Test your changes in a real WSL + Windows environment if possible.
- Follow existing formatting and structure.

## Reporting issues

Open a GitHub Issue describing the problem, your environment (Windows version, WSL distro), and steps to reproduce.
